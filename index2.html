Atom: A software to write code and make changes. First, you download Atom from the website atom.io.

Next, you download the packages you need:
  • atom-beautify
  • pigments
  • open-in-browser
  • autoclose-html
  • linter-jshint
  • linter-csslint
  • atom-bootstrap3

    • Themes you should select in Atom:

  • monokai-seti
  • seti-iu

    • You must restart the program for these changes to work.

    Github: a software to place files and folders by syncing files them in different computers. You must make a repository which you should not edit from.
    Through github you may also view the pages you are working on. However, it is mainly a tool to hold your files.

    Github for desktop: This tool allows you to clone your git and look at changes and decide if you would like to commit to them. Once you are okay with changes,
    you sync them. This should be done only when you need to.
